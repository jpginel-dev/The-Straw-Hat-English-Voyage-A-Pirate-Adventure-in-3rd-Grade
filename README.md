# The-Straw-Hat-English-Voyage-A-Pirate-Adventure-in-3rd-Grade
"The Straw Hat English Voyage" es una Situación de Aprendizaje (SdA) gamificada para la asignatura de Inglés en 3º de Educación Primaria. Utiliza el universo del manga/anime One Piece como un hilo conductor motivador para el desarrollo de destrezas lingüísticas comunicativas iniciales
